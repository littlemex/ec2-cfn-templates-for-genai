#### 2. Tangent Mode№╝ѕУёЄжЂЊсЃбсЃ╝сЃЅ№╝Ѕ

**сѓ│сЃъсЃ│сЃЅ**: `/tangent` сЂЙсЂЪсЂ» **Ctrl+T**

- сЃАсѓцсЃ│С╝џУЕ▒сЂ«сѓ╣сЃгсЃЃсЃЅсЂФТѕ╗сѓІ

**тѕЕуѓ╣**:

- ТЌбтГўсЂ« TODO сЃфсѓ╣сЃѕсЂ«тєЇжќІ

#### 5. Checkpointing№╝ѕсЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕ№╝Ѕ

**сѓ│сЃъсЃ│сЃЅ**: `/checkpoint`

Git сЃЕсѓцсѓ»сЂфсѓ│сЃъсЃ│сЃЅсѓњСй┐ућесЂЌсЂдсђЂсЃЋсѓАсѓцсЃФтцЅТЏ┤сѓњУ┐йУиАсЂЎсѓІсѓ╗сЃЃсѓисЃДсЃ│сѓ╣сѓ│сЃ╝сЃЌсЂ«сЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕТЕЪУЃйсЂДсЂЎсђѓ

**СИ╗сЂфТЕЪУЃй**:

- сЃЋсѓАсѓцсЃФтцЅТЏ┤сѓњсѓисЃБсЃЅсѓд Git сЃфсЃЮсѓИсЃѕсЃфсЂФсѓ╣сЃісЃЃсЃЌсѓисЃДсЃЃсЃѕ
- сЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕсЂ«сЃфсѓ╣сЃѕУАеуц║сђЂУЕ│у┤░УАеуц║сђЂти«тѕєУАеуц║сђЂтЙЕтЁЃ
- тЙЕтЁЃТЎѓсЂФС╝џУЕ▒т▒ЦТГ┤сѓњти╗сЂЇТѕ╗сЂЌ
- Git сЃфсЃЮсѓИсЃѕсЃфсЂДУЄфтІЋТюЅті╣тїќ
- жЮъ Git сЃЄсѓБсЃгсѓ»сЃѕсЃфсЂДсЂ»ТЅІтІЋтѕЮТюЪтїќ

**ТюЅті╣тїќТќ╣Т│Ћ**:

```bash
kiro-cli settings chat.enableCheckpoint true
```

**тЪ║ТюгуџёсЂфСй┐ућеТќ╣Т│Ћ**:

```bash

# сЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕсЂ«УЕ│у┤░ТЃЁта▒сѓњУАеуц║

# сЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕжќЊсЂ«ти«тѕєсѓњТ»ћУ╝Ѓ

# сЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕсЂФтЙЕтЁЃ
/checkpoint restore [<tag>]

# сѓ╗сЃЃсѓисЃДсЃ│сЂ«сѓисЃБсЃЅсѓдсЃфсЃЮсѓИсЃѕсЃфсѓњтЅіжЎц
```

**сЃдсЃ╝сѓ╣сѓ▒сЃ╝сѓ╣**:

- т«ЪжеЊуџёсЂфтцЅТЏ┤сѓњУЕдсЂЎтЅЇсЂФсѓ╗сЃ╝сЃЋсЃЮсѓцсЃ│сЃѕсѓњСйюТѕљ

- тцЅТЏ┤т▒ЦТГ┤сЂ«У┐йУиАсЂеу«Ауљє
- тЋЈжАїсЂїуЎ║ућЪсЂЌсЂЪжџЏсЂ«т«ЅтЁесЂфтЙЕтЁЃ

#### 7. Delegate№╝ѕтДћУГ▓№╝Ѕ

жЮътљїТюЪсѓ┐сѓ╣сѓ»сЃЌсЃГсѓ╗сѓ╣сѓњУхитІЋсЃ╗у«АуљєсЂЌсђЂуЅ╣т«џсЂ«сѓесЃ╝сѓИсѓДсЃ│сЃѕсЂеСИдтѕЌсЂДKiro сЃЂсЃБсЃЃсЃѕсѓ╗сЃЃсѓисЃДсЃ│сѓњт«ЪУАїсЂЌсЂЙсЂЎсђѓ

**СИ╗сЂфТЕЪУЃй**:

- УЄфуёХУеђУфъсѓњСй┐ућесЂЌсЂдсЃљсЃЃсѓ»сѓ░сЃЕсѓдсЃ│сЃЅсѓ┐сѓ╣сѓ»сѓњУхитІЋ
- уЅ╣т«џсЂ«сѓесЃ╝сѓИсѓДсЃ│сЃѕсЂеСИдтѕЌсЃЂсЃБсЃЃсЃѕсѓ╗сЃЃсѓисЃДсЃ│сѓњт«ЪУАї
- сѓ┐сѓ╣сѓ»сЂ«жђ▓ТЇЌсѓњуІгуФІсЂЌсЂдуЏБУдќ
- сѓ╗сѓГсЃЦсЃфсЃєсѓБсЂ«сЂЪсѓЂсЂ«сѓесЃ╝сѓИсѓДсЃ│сЃѕТЅ┐УфЇсЃЋсЃГсЃ╝

**ТюЅті╣тїќТќ╣Т│Ћ**:

```bash
kiro-cli settings chat.enableDelegate true
```

**Сй┐ућеТќ╣Т│Ћ**:

УЄфуёХУеђУфъсЂД Kiro сЂФсЃљсЃЃсѓ»сѓ░сЃЕсѓдсЃ│сЃЅсѓ┐сѓ╣сѓ»сЂ«УхитІЋсѓњСЙЮжа╝сЂЌсЂЙсЂЎ№╝џ

```
API сѓесЃ│сЃЅсЃЮсѓцсЃ│сЃѕсЂ«сЃЉсЃЋсѓЕсЃ╝сЃъсЃ│сѓ╣сѓњтѕєТъљсЂЎсѓІсЃљсЃЃсѓ»сѓ░сЃЕсѓдсЃ│сЃЅсѓ┐сѓ╣сѓ»сѓњСйюТѕљсЂЌсЂдсЂЈсЂасЂЋсЂё
```

сЂЮсЂ«тЙїсђЂухљТъюсѓњуб║УфЇ№╝џ

```
API тѕєТъљсѓ┐сѓ╣сѓ»сЂ«сѓ╣сЃєсЃ╝сѓ┐сѓ╣сѓњуб║УфЇсЂЌсЂдсЂЈсЂасЂЋсЂё
сЃљсЃЃсѓ»сѓ░сЃЕсѓдсЃ│сЃЅтѕєТъљсЂ«ухљТъюсѓњУдІсЂЏсЂдсЂЈсЂасЂЋсЂё
```

**сЃдсЃ╝сѓ╣сѓ▒сЃ╝сѓ╣**:

- жЋиТЎѓжќЊсЂІсЂІсѓІтѕєТъљсѓ┐сѓ╣сѓ»сЂ«СИдтѕЌт«ЪУАї
- УцЄТЋ░сЂ«сѓ│сЃ╝сЃЅсЃЎсЃ╝сѓ╣Уф┐ТЪ╗сЂ«тљїТЎѓжђ▓УАї
- сЃАсѓцсЃ│сѓ┐сѓ╣сѓ»сѓњСИГТќГсЂЏсЂџсЂФсЃљсЃЃсѓ»сѓ░сЃЕсѓдсЃ│сЃЅсЂДТЃЁта▒тЈјжЏє
- УцЄТЋ░сЂ«сѓесЃ╝сѓИсѓДсЃ│сЃѕсѓњТ┤╗ућесЂЌсЂЪті╣ујЄуџёсЂфжќІуЎ║

### сЃЋсѓАсѓИсЃ╝Тцюу┤бсѓхсЃЮсЃ╝сЃѕ

сЂЎсЂ╣сЂдсЂ«т«ЪжеЊуџёсѓ│сЃъсЃ│сЃЅсЂ»сђЂсЃЋсѓАсѓИсЃ╝Тцюу┤б№╝ѕCtrl+S№╝ЅсЂДтѕЕућетЈ»УЃйсЂДсЂЎ№╝џ

- `/todo` - TODO сЃфсѓ╣сЃѕсѓ│сЃъсЃ│сЃЅ№╝ѕТюЅті╣ТЎѓ№╝Ѕ
- `/tangent` - Tangent Mode сЂ«тѕЄсѓіТЏ┐сЂѕ№╝ѕТюЅті╣ТЎѓ№╝Ѕ
- `/checkpoint` - сЃЂсѓДсЃЃсѓ»сЃЮсѓцсЃ│сЃѕсѓ│сЃъсЃ│сЃЅ№╝ѕТюЅті╣ТЎѓ№╝Ѕ

### сЃЎсѓ╣сЃѕсЃЌсЃЕсѓ»сЃєсѓБсѓ╣

1. **т«ЅтЁесЂфуњ░тбЃсЂДсЃєсѓ╣сЃѕ**: сЂЙсЂџжЄЇУдЂт║дсЂ«СйјсЂёсЃЌсЃГсѓИсѓДсѓ»сЃѕсЂДт«ЪжеЊуџёТЕЪУЃйсѓњУЕдсЂЌсЂдсЂЈсЂасЂЋсЂё
2. **сЃЋсѓБсЃ╝сЃЅсЃљсЃЃсѓ»сЂ«ТЈљСЙЏ**: `kiro issue` сѓњСй┐ућесЂЌсЂдтЋЈжАїсѓёТЈљТАѕсѓњта▒тЉісЂЌсЂдсЂЈсЂасЂЋсЂё
3. **ТюђТќ░ТЃЁта▒сЂ«уб║УфЇ**: сЃфсЃфсЃ╝сѓ╣сЃјсЃ╝сЃѕсЂДт«ЪжеЊуџёТЕЪУЃйсЂ«тцЅТЏ┤сѓњуб║УфЇсЂЌсЂдсЂЈсЂасЂЋсЂё

5. **сЃљсЃЃсѓ»сѓбсЃЃсЃЌсЂ«СйюТѕљ**: СИђжЃесЂ«ТЕЪУЃйсЂ»сЃЋсѓАсѓцсЃФсѓњтцЅТЏ┤сЂЌсЂЙсЂЎ№╝ѕCheckpointingсђЂTODO Lists№╝Ѕ
